# 参考文件：美国膳食指南 DGA 2025-2030

> 来源：Dietary Guidelines for Americans, 2025–2030
> 发布：2026年1月7日 | 官网：https://www.dietaryguidelines.gov/
> 说明：每五年更新一次。本版较 DGA 2020-2025 有较大调整，引发学界争议，仅供参考。

---

## 核心推荐（学界共识部分）

| 推荐项 | 具体内容 |
|--------|---------|
| 蔬菜水果 | 增加摄入量，多样化选择 |
| 全谷物 | 优先选择全谷物替代精制谷物 |
| 添加糖 | 限制摄入，不超过总热量的 10% |
| 钠 | 避免过量摄入，<2300毫克/天 |
| 高度加工食品 | 减少摄入（高添加糖、钠、饱和脂肪的垃圾食品） |

---

## 关键营养素目标

| 营养素 | 目标 | 说明 |
|--------|------|------|
| 膳食纤维 | 25-34克/天 | 增加蔬果和全谷物摄入 |
| 钙 | 1000-1300毫克/天 | 低脂/脱脂奶制品、豆制品、深绿色蔬菜 |
| 维生素D | 600-800 IU/天 | 日照、强化食品、补充剂 |
| 钠 | <2300毫克/天 | 高血压者 <1500毫克/天 |
| 添加糖 | <10%总热量 | 约 50克/天（以 2000千卡计） |
| 饱和脂肪 | <10%总热量 | 用不饱和脂肪替代 |
| 钾 | 4700毫克/天 | 预防高血压 |

## 2000千卡健康饮食模式示例

| 食物类别 | 日推荐量 | 备注 |
|---------|---------|------|
| 蔬菜 | 2.5杯当量 | 深绿色、红色/橙色、豆类、淀粉类、其他 |
| 水果 | 2杯当量 | 优先新鲜水果 |
| 全谷物 | 至少一半谷物为全谷物 | — |
| 精制谷物 | 不超过一半 | — |
| 乳制品 | 3杯当量 | 低脂/脱脂优先 |
| 蛋白质食物 | 5.5盎司当量 | 瘦肉、禽肉、鱼、蛋、豆类、坚果 |
| 油脂 | 27克 | 优先不饱和脂肪酸来源 |
| 添加糖上限 | <50克 | 约 12 茶匙 |
| 饱和脂肪上限 | <22克 | — |
| 钠上限 | <2300毫克 | — |

## 与 DGA 2020-2025 的主要变化及争议

| 变化点 | 旧版 | 新版 | 争议说明 |
|--------|------|------|---------|
| 蛋白质 | 适量 | **强化强调** | 学界认为多数美国人已充足，过度强调可能增加饱和脂肪摄入 |
| 纤维 | 明确强调 | **弱化** | 纤维是美国人群最缺乏的营养素之一 |
| 饱和脂肪 | <10%总热量 | **保留限值** | 但同步推荐红肉/全脂奶，内部矛盾 |
| 加工食品术语 | 明确 | **模糊化** | 给食品行业留下可操作空间 |
| 乳制品 | 推荐 | **继续强化** | 全球 75% 人口乳糖不耐受 |
| 添加糖 | <10%总热量 | 保留 | ✅ 无争议 |
| 钠 | <2300毫克 | 保留 | ✅ 无争议 |

## 中国人群适用调整建议

| DGA推荐 | 中国人群调整 |
|---------|------------|
| 乳制品 3杯/天 | 调整为 1-2杯 + 豆制品补充钙（乳糖不耐受普遍） |
| 饱和脂肪 <10% | 保留，中式餐饮总体达标 |
| 添加糖 <50克 | 保留，中式餐饮总体达标 |
| 钠 <2300毫克 | **超目标**，需特别注意控盐（中式餐饮盐分偏高） |
| 膳食纤维 25-34克 | 保留，中式餐饮若搭配粗粮可达标 |
| 蛋白质强调 | 不必照搬高蛋白目标，按中国指南15-20%供能比即可 |
