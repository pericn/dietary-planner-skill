# 参考文件：成人肥胖食养指南（2024年版）

> 来源：国家卫生健康委办公厅《成人肥胖食养指南（2024年版）》
> 发布：2024年 | 适用对象：成人肥胖患者（也可作为一般减重参考）

---

## 一、肥胖判定标准

### BMI 分类

| 分类 | 标准 |
|------|------|
| 正常 | 18.5 ≤ BMI < 24.0 |
| 超重 | 24.0 ≤ BMI < 28.0 |
| **肥胖** | **BMI ≥ 28.0** |

### 腰围（中心型肥胖）

| 分类 | 男性 | 女性 |
|------|------|------|
| 中心型肥胖前期 | 85cm ≤ WC < 90cm | 80cm ≤ WC < 85cm |
| **中心型肥胖** | **WC ≥ 90cm** | **WC ≥ 85cm** |

## 二、能量控制目标

### 减重能量目标

| 方法 | 目标 |
|------|------|
| 每日减少幅度 | 降低 500 ~ 1000千卡/天（或降低 30% ~ 50%） |
| 男性限能量 | 1200 ~ 1500千卡/天 |
| 女性限能量 | 1000 ~ 1200千卡/天 |

### 宏量营养素供能比

| 营养素 | 供能占比 |
|--------|---------|
| 脂肪 | 20% ~ 30% |
| 蛋白质 | 15% ~ 20% |
| 碳水化合物 | 50% ~ 60% |

### 三餐供能比

| 餐次 | 供能占比 |
|------|---------|
| 早餐 | 约 30% |
| 午餐 | 约 40% |
| 晚餐 | 约 30% |

### 控盐控油控糖

| 项目 | 目标 |
|------|------|
| 食盐 | < 5克/天 |
| 烹调油 | 20 ~ 25克/天 |
| 添加糖 | < 25克/天 |
| 饮水 | 1500 ~ 1700毫升/天（7~8杯） |

## 四、中医辨证分型与食药物质

| 证型 | 症状概要 | 推荐食药物质 |
|------|---------|------------|
| **胃热火郁证** | 多食易饥，大便干结，口干口苦 | 铁皮石斛、麦芽、鲜芦根、荷叶 |
| **痰湿内盛证** | 形体肥胖，身体沉重，脘痞胸满 | 薏苡仁、橘皮、砂仁、茯苓、白扁豆 |
| **气郁血瘀证** | 肥胖懒动，喜太息，面晦唇暗 | 山楂、桃仁、佛手 |
| **脾虚不运证** | 肥胖臃肿，神疲乏力，四肢浮肿 | 黄芪、山药、莲子、白扁豆、茯苓 |
| **脾肾阳虚证** | 形体肥胖，四肢不温，喜食热饮 | 山药、黄芪、肉桂、干姜 |

## 五、饮食行为建议

1. **定时定量规律进餐** — 重视早餐，不漏餐；晚餐建议 17:00~19:00 进食
2. **晚餐后不再进食** — 可饮水，不吃任何食物
3. **少吃零食，少喝饮料** — 控制随意进食，避免夜宵
4. **细嚼慢咽** — 减慢进餐速度，增加饱腹感
5. **改变进餐顺序** — 蔬菜 → 肉类 → 主食（减少高能量食物摄入量）
6. **控制外卖和外出就餐频率**

## 六、运动建议

### 运动方案

| 运动类型 | 频率 | 时长/强度 |
|---------|------|----------|
| 有氧运动（中低强度） | 每周 5~7 天，至少隔天1次 | 每周 150~300 分钟 |
| 抗阻运动 | 每周 2~3 天，隔天1次 | 每次 10~20 分钟 |
| 每周通过运动消耗能量 | — | ≥ 2000千卡 |

### 减少久坐

- 每天静坐和被动视屏时间控制在 2~4 小时以内
- 长期伏案者，每小时起来活动 3~5 分钟

### 睡眠

- 保证每日 7 小时左右睡眠时间
- 按昼夜生物节律作息，避免熬夜
- 睡眠不足可导致内分泌紊乱、脂肪代谢异常（"过劳肥"）

## 七、地区特色食谱说明

本指南按 7 大地区提供食谱示例，各有针对性：

| 地区 | 饮食特点 | 减重重点 |
|------|---------|---------|
| 东北 | 主食多面食，炖菜多 | 控制油盐，增加蔬菜比例 |
| 西北 | 喜食牛羊肉和面食 | 控制红肉量，增加绿叶蔬菜 |
| 华北 | 面食为主，蔬菜相对少 | 增加蔬菜和水果多样性 |
| 华东 | 清淡偏甜，海鲜多 | 控糖，优选鱼虾类蛋白 |
| 华中 | 口味偏辣，油盐重 | 重点控油控盐（适合湘菜家庭参考） |
| 西南 | 麻辣为主，油脂偏高 | 减少烹调用油，增加蒸煮比例 |
| 华南 | 清淡偏甜，水产多 | 控糖，充分利用水产蛋白 |

> 注：本技能的目标家庭以湘菜/华中口味为主，应重点参考华中和西南地区的油盐控制方案。
